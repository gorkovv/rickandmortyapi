## Описание проекта

Мобильное приложение на **Flutter**, отображающее список персонажей из вселенной **Rick & Morty** с использованием открытого API.

Основные возможности приложения:

- загрузка и отображение списка персонажей;
- постраничная подгрузка данных (pagination);
- добавление и удаление персонажей из **избранного**;
- сохранение избранных персонажей локально (кэш);
- экран **избранных персонажей**;
- анимированное удаление карточек из списка;
- сортировка избранных персонажей:
    - по имени
    - по статусу (живой, мёртвый, неизвестно)
    - по виду (человек, инопланетянин и т.д.)
- переключение **светлой и тёмной темы**;
- нижняя навигация между экранами:
    - Персонажи
    - Избранное

Архитектура приложения построена с использованием:

- **Clean Architecture**
- **BLoC / Cubit (flutter_bloc)**
- **Repository pattern**

## Сборка и запуск проекта

### 1. Клонировать репозиторий
```bash
git clone <repository_url>
cd <project_folder>
```
### 2. Установить зависимости
```bash
flutter pub get
```
### 3. Запустить приложение
```bash
flutter run
```
